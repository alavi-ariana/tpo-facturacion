# Sistema de Facturación - TPO Ingeniería de Datos II

Relational database implementation (PostgreSQL) for a billing system.

## Prerequisites
- Docker
- Docker Compose

## Project Structure
- `/data`: Source CSV files.
- `/scripts/postgres`: SQL scripts organized by DDL, DML, and queries.
- `/scripts/mongo`: JavaScript scripts for MongoDB collections and queries.
- `docker-compose.yml`: Container orchestration for PostgreSQL and pgAdmin.
